# California Housing Prices - Predicción de Precios de Viviendas

Análisis predictivo del valor medio de viviendas en California utilizando machine learning.

## Dataset
Contiene información sobre:
- Ubicación geográfica (longitud, latitud)
- Características de las viviendas (edad, habitaciones, dormitorios)
- Datos demográficos (población, hogares, ingresos)
- Variable objetivo: `median_house_value`

## Tecnologías
- Python
- Pandas, NumPy
- Scikit-learn
- Matplotlib, Seaborn
- Múltiples algoritmos de ML

## Objetivo
Desarrollar modelos de regresión para predecir el valor medio de las viviendas basándose en características económicas y demográficas.

## Métodos
- Preprocesamiento y feature engineering
- Múltiples modelos (Linear Regression, Random Forest, SVM, etc.)
- Evaluación con métricas: MAE, MSE, R²
- Optimización con GridSearchCV


Análisis completo de precios de viviendas en California con implementación de pipelines de ML y comparativa de modelos.
